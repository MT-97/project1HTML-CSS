Opdracht 2.3 onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

De index pagina kan ook wel de homepage genoemd worden,de reden dat het index wordt genoemd is omdat het een index bestand automatisch wordt geopend

Met welke HTML versie werken we nu?

De huidige versie is HTML 5. Dit is ook de nieuwste versie die gebruikt word

Met welke CSS versie werken we nu?

moderne browsers ondersteunen CSS versie 1 en versie 2. Er is wel al een CSS versie 3 maar nog niet alle browsers ondersteunen deze versie.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Een website kan er anders uitzien in verschillende browsers omdat elke browser de code van de website op een andere manier interpreteert. Sommige browsers ondersteunen bepaalde codes of effecten niet, of geven ze anders weer dan andere browsers.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populairste browser die gebruikt word is Chrome. Dit komt door de snelheid en het uiterlijk.Een artikel uit 2021 meldt dat Chrome gemiddeld 1,77 GB RAM gebruikt per tabblad
